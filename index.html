Reactive - JavaScript template engine for realtime applications

Reactive

Reactive is a simple yet robust view engine for real-time "reactive" applications. Reactive views are live DOM, no static compilation of strings involved, just fast, flexible, and robust DOM-based views that react to your models.

  • Text bindings

    Through interpolation you may bind to properties and Reactive will adjust the contents upon changes.

  • Hello {name}

  • Hello {name}

  • Complex expressions

    With Reactive the full range of JavaScript expressions are at your disposal. This means no learning a new language or subset, it's just js! As you can see Reactive still has no problem picking out which properties to bind.

  • Hello { name || 'Nobody' }

  • Hello { name || 'Nobody' }
